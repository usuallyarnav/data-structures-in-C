# Unit 1 — Searching, Sorting & Algorithm Basics

This unit covers the foundational building blocks of programming — how to search for data, sort it efficiently, and analyze how well your code performs.

---

## 🔍 Searching

| File | Description |
|------|-------------|
| `linear_search.c` | Searches for an element by checking each item one by one from the beginning. |
| `binary_search.c` | Searches for an element in a sorted array by repeatedly halving the search range. |

---

## 🔃 Sorting

| File | Description |
|------|-------------|
| `bubble_sort.c` | Sorts an array by repeatedly swapping adjacent elements that are in the wrong order. |
| `insertion_sort.c` | Builds a sorted array one element at a time by inserting each into its correct position. |
| `selection_sort.c` | Sorts by repeatedly finding the smallest element and placing it at the beginning. |
| `radix_sort.c` | Sorts numbers digit by digit, from the least significant to the most significant. |
| `quick_sort.c` | Picks a pivot element and partitions the array around it, then sorts each partition recursively. |
| `heap_sort.c` | Sorts by building a heap from the array and repeatedly extracting the largest element. |
