# Unit 2 — Linked Lists

This unit covers linked lists — a fundamental data structure where elements are connected through pointers rather than stored in contiguous memory like arrays.

---

## 🔗 Linked Lists

| File | Description |
|------|-------------|
| `singly_linked_list.c` | A list where each node points to the next one. Covers insertion and deletion. |
| `circular_linked_list.c` | A singly linked list where the last node points back to the first, forming a circle. |
| `doubly_linked_list.c` | A list where each node points to both the next and the previous node, allowing traversal in both directions. |
